### Iter 12.5 — Trial-til-betalt konvertering med korrekt fakturadato (D-045 + D-049)
```

     bruker IKKE blir dobbeltbetalt for resterende trial-dager. Customer
     opprettes just-in-time (D-049) — aldri ved trial-registrering.

Endepunkt: POST /api/billing/create-checkout
Input: { plan: "monthly" | "yearly" }
Identitet: subdomain fra host-header (D-046)

Felles flyt (alle 3 scenarier):
    1. Hent TenantRecord fra sentral Upstash (host-header → subdomain)
    2. Just-in-time Stripe customer (per D-049):
        if (!tenant.stripeCustomerId) {
          const customer = await stripe.customers.create({
            email: tenant.email,
            name: `${tenant.firstName} ${tenant.lastName}`,
            metadata: { subdomain: tenant.subdomain }
          });
          await updateTenant(subdomain, { stripeCustomerId: customer.id });
        }
    3. Velg scenario basert på status + trialEndsAt
    4. Opprett Stripe Checkout session
    5. Returner { url } — klienten redirecter

Logikk for å velge scenario:
    if (tenant.status === "trial" && now < trialEndsAt) → Scenario A
    if (tenant.status === "locked")                     → Scenario B
    if (tenant.status === "pending" && plan !== "trial") → Scenario C

Scenario A — Trial-bruker konverterer (dag 1-29):
    stripe.checkout.sessions.create({
      customer: stripeCustomerId,
      mode: "subscription",
      line_items: [{ price: priceId, quantity: 1 }],
      subscription_data: {
        trial_end: Math.floor(
          new Date(tenant.trialEndsAt).getTime() / 1000
        )
      },
      success_url: `https://${subdomain}.kodovault.no/billing/success`,
      cancel_url: `https://${subdomain}.kodovault.no`
    })
    → Første faktura: eksakt på trialEndsAt-datoen
    → Neste faktura: trialEndsAt + 30d (monthly) / + 365d (yearly)

Scenario B — Trial utløpt, bruker møter betalingsvegg (dag 30+):
    Samme som A men UTEN subscription_data.trial_end
    → Første faktura: umiddelbart
    → Neste faktura: i dag + 30d / + 365d

Scenario C — Ny bruker velger betalt plan direkte (dag 0, fra Iter 12):
    stripe.checkout.sessions.create({
```
