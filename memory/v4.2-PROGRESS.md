### sv.json + da.json sync
**BYTE-LIKT mønster** — etter hver no.json-endring:
```python
import re
with open('no.json') as f: no_text = f.read()
meta_pattern = re.compile(r'(\{\n  "_meta": \{[^}]+\},\n)', re.DOTALL)
sv_meta = '{...sv-_meta...},\n'
da_meta = '{...da-_meta...},\n'
sv_text = meta_pattern.sub(sv_meta, no_text, count=1)
da_text = meta_pattern.sub(da_meta, no_text, count=1)
```
Verifiser: `diff no.json sv.json` skal kun vise `_meta`-blokken (8 linjer ulike: 3 verdier + omkringliggende JSON-struktur).

```python
import re
with open('no.json') as f: no_text = f.read()
meta_pattern = re.compile(r'(\{\n  "_meta": \{[^}]+\},\n)', re.DOTALL)
sv_meta = '{...sv-_meta...},\n'
da_meta = '{...da-_meta...},\n'
sv_text = meta_pattern.sub(sv_meta, no_text, count=1)
da_text = meta_pattern.sub(da_meta, no_text, count=1)
```
Verifiser med: `diff no.json sv.json` skal kun vise `_meta`-blokken (3 linjer ulike).
